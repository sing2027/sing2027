HTML Tags

<h1> - <h6>
Heading tags are used to create different levels of headings.
<p>
Paragraph tag is used to define a paragraph of the text.
<ul> and <li>
Unordered list tag and list item tag are used to create bulleted lists.
<a>
Anchor tag is used to create a hyperlink to another web page.
<img>
Image tag used to display images on a web page.
